# 🐰 ASCII 토끼가 ASCII 당근을 먹는 모션 구현 명세

본 문서는 **제공된 영상 및 이미지 기준**으로,
토끼와 당근을 **모두 ASCII 문자로만 표현**하여 애니메이션을 구현하기 위한
**IDE AI 툴 전달용 명확한 구현 명세서**입니다.

> ❗ 핵심 원칙
>
> -   이모지, 이미지, SVG 사용 금지
> -   모든 요소는 고정폭 문자(ASCII)만 사용
> -   `<pre>` / 콘솔 / 텍스트 렌더링 기준

---

## 1. 구현 목표

-   토끼는 ASCII 아트로 고정 위치에 표시
-   당근은 아래에서 위로 이동
-   당근이 입에 닿으면:

    -   당근 제거
    -   토끼 입이 씹는 모션으로 변경

-   전체 애니메이션은 루프 가능

---

## 2. 토끼 ASCII 정의 (고정)

```text
  ||  ||
 ( o  )
(   ᵔ  )
 (      )
```

-   토끼는 **항상 같은 위치**에 렌더링
-   눈과 입은 상태에 따라 일부 문자만 변경

### 2.1 입 상태 정의

| 상태   | 표현    |
| ------ | ------- |
| 기본   | `ᵔ`     |
| 씹기 1 | `>` `<` |
| 씹기 2 | `^` `^` |

---

## 3. 당근 ASCII 정의 (이미지 기준)

> 이미지에 표시된 ↓ / o / v 조합을 그대로 사용

### 기본 당근

```text
  ↓
  o
  v
```

### 확장형 (권장)

```text
  ↓
  |
  o
  v
```

-   ↓ : 잎
-   o : 몸통
-   v : 뿌리

## 6. 구현 가이드 (IDE AI 툴용)

-   모든 프레임은 **문자열 배열**로 관리
-   프레임 전환은 타이머 기반

    -   권장 간격: 100~150ms

-   렌더링은 다음 중 하나

    -   콘솔 출력 (`console.clear()` + `console.log()`)
    -   웹 `<pre>` 태그

### 렌더링 조건

```text
font-family: monospace
white-space: pre
line-height: 1.1
```
